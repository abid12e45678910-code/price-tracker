# 📚 Book Price Tracker

This project is a Python script that scrapes book prices from a website and stores them in a CSV file.

## 🔧 Technologies Used
- Python
- requests
- BeautifulSoup
- csv

## 🚀 Features
- Scrapes book titles and prices
- Stores data in CSV format
- Simple and easy to use

## ▶️ How to Run

1. Install dependencies:
   pip install requests beautifulsoup4

3. Run the script:
   python price_tracker.py
